# 知识库问答统计技能

## 概述

查询知识库的问答统计数据，帮助管理员了解知识库的使用情况。

## 核心能力

- 统计指定时间范围内的问答数量
- 展示热门问题排行榜
- 提供响应时间等性能指标
- 支持按时间维度查看每日详情

## 时间范围约定

| 参数值 | 含义 |
|--------|------|
| 7d | 最近 7 天 |
| 30d | 最近 30 天（默认） |
| 90d | 最近 90 天 |
| 1y | 最近 1 年 |

## 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| timeRange | string | 查询时间范围 |
| topN | integer | 返回前 N 条热门问题 |
| includeDetails | boolean | 是否包含每日详情 |

## 工作流程

1. **确定时间范围**：根据用户描述解析 timeRange
2. **确定数量限制**：根据用户描述解析 topN
3. **判断详情需求**：根据用户描述判断 includeDetails
4. **调用工具**：使用 knowledge_stats_query 执行查询
5. **格式化输出**：生成 Markdown 格式报告

## 使用示例

### 示例 1：热门问题统计

用户问："查看最近一周的热门问题统计"

参数提取：
- timeRange: "7d"
- topN: 10
- includeDetails: false

工具调用：
```
knowledge_stats_query(timeRange="7d", topN=10, includeDetails=false)
```

输出：
```
## 知识库问答统计

**查询时间**: 最近 7 天

### 概览
- 总问答数: 1,258
- 总访问量: 4,567
- 平均响应时间: 235ms
- 成功率: 96.8%

### 热门问题 TOP 10
1. 如何申请年假？(892 次)
2. 请假审批流程是什么？(756 次)
...
```

### 示例 2：带详情的统计

用户问："给我过去一个月访问量最高的前20个问题，要包含详细信息"

参数提取：
- timeRange: "30d"
- topN: 20
- includeDetails: true

## 注意事项

1. 只允许管理员角色调用
2. 查询时间范围不能超过 1 年
3. topN 最大支持 100 条
4. includeDetails=true 时会返回每日明细，数据量较大

## 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 返回空数据 | 时间范围内无数据 | 扩大 timeRange |
| 数据明显偏少 | 只统计了部分时间 | 检查时间范围边界 |
| 热门问题重复 | 未做去重处理 | 使用内容哈希去重 |
