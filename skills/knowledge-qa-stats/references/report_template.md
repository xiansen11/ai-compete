# 知识库问答统计报告模板

## 报告结构

### 概览部分
- 总问答数：时间段内的问答总条数
- 总访问量：独立会话数
- 平均响应时间：所有成功请求的平均处理时间(ms)
- 成功率：成功请求占总请求的比例

### 热门问题部分
按查询次数降序排列，显示问题内容和访问次数

### 每日详情（可选）
当 includeDetails=true 时显示，按日期倒序排列

## 数据来源

数据来源于以下数据库表：
- conversation_message：问答消息记录
- rag_trace_run：RAG 追踪运行记录

## 统计口径

- 问答数：assistant角色的消息数量
- 访问量：独立会话数
- 热门问题：user角色的消息，按内容分组计数
